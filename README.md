# Praxys

Praxys is a productivity and execution platform designed to help users turn intentions into consistent action through focused work sessions, task management, streak tracking, and progress analytics.

## Vision

Most productivity tools help users plan. Praxys focuses on execution.

The goal is to provide a structured system that helps users stay focused, build discipline, and make measurable progress toward their goals.

## Core Features

* Task Management
* Focus Sessions
* Streak Tracking
* Productivity Analytics
* Progress Dashboard
